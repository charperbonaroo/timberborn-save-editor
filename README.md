# Timberborn Save Editor

## Installation

Use NodeJS 12.13.1. Other versions are untested. Yes, it's old.

```sh
npm i -g yarn
yarn install
```

## Starting

```sh
yarn start
```

## Test / CI

There are no tests, but build needs to succeed.

```sh
yarn build
```

## Deploy

Ensure you're on master branch and there are no changes. Then:

```sh
yarn deploy
```
